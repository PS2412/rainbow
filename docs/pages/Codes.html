  Class Codes

This class contains all the color codes used for rainbow class.

Since

1.0

Package org.techous.rainbow
class  Codes
extends  java.lang.Object

Field Details

These are all the constants used in the java library which you cannot use while calling any function. For every field, codes are given with pics which you can use only while calling rainbow functions.
